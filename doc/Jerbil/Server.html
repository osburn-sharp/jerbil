get_local_server(servers) click to toggle source

returns the local server from an array of servers. Aimed at locating the local server to use from the output of the config file

# File lib/jerbil/server.rb, line 27
def self.get_local_server(servers)
  hostname = Socket.gethostname
  servers.each {|server| return server if server.fqdn == hostname}
end

# File lib/jerbil/server.rb, line 32
def initialize(fqdn, key, port=nil)
  @fqdn = fqdn
  @key = key
  @port = port || Socket.getservbyname('jerbil')
  @active = false
end
